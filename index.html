Hotspot 1 (Windows Server – AD/GPO): identidad, usuarios, grupos, políticas.
Hotspot 2 (Linux Server – Web/DNS): servicios de infraestructura (web/DNS) y operación.
Hotspot 3 (Seguridad/Continuidad): firewall, monitoreo, logs y backups.
